### ¿Cuál de los siguientes esquemas provee tolerancia a fallas en sistemas de almacenamiento?
**Sistema RAID**
**RAID** (_Redundant Array of Independent Disks_) es una tecnología diseñada específicamente para **sistemas de almacenamiento**. 

Permite combinar varios discos rígidos físicos en una sola unidad lógica, distribuyendo los datos o replicándolos (mediante espejado o paridad) de modo que, si un disco físico falla, los datos no se pierdan y el sistema siga operativo.

### ¿Cuál de las siguientes estrategias no tendría sentido en un ataque CSRF?
**Banear la IP de quien dispara el request**

En un ataque **CSRF** (_Cross-Site Request Forgery_), un sitio malicioso logra que el navegador de una víctima legítima envíe una petición no deseada a una aplicación web donde la víctima ya está autenticada.
